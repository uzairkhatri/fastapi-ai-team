# Code of Conduct

## Our standards

This is a technical project. Interactions should be direct, specific, and constructive.

Expected behavior:
- Be specific in issues and PRs — include examples, reproduction steps, or sample output
- Critique the work, not the person
- Accept that maintainers may close issues or reject PRs without extensive justification

Unacceptable behavior:
- Harassment, personal attacks, or discriminatory language
- Spamming issues or PRs with vague requests
- Demanding support or timelines

## Enforcement

Violations may result in comments being removed or contributors being blocked from the repository. There is no formal appeal process.

## Scope

This applies to all project spaces — GitHub issues, PRs, Discussions, and any other venue where this project is discussed with reference to this repo.

## Contact

Report conduct issues to the maintainer via [GitHub Discussions](https://github.com/uzairkhatri/fastapi-ai-team/discussions) or by mentioning [@uzairkhatri](https://github.com/uzairkhatri) in an issue.
